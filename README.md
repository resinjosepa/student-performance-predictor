# 🎓 Student Performance Predictor

A Machine Learning web application built with **Flask** that predicts a student's final academic performance using a **Random Forest Regression** model.

---

## 🚀 Features

- 🔐 User Authentication (Login & Registration)
- 📊 Student Performance Prediction
- 🤖 Random Forest Machine Learning Model
- 📝 Multi-Step Prediction Form
- 💾 SQLite Database
- 🎨 Modern Dark SaaS UI
- 📱 Responsive Design

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- Scikit-learn
- Pandas
- NumPy
